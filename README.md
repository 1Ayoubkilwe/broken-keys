## Pluses and Hashes

##### Your Task 📚:

Create a function called `plusHashCount` that returns the number of hashes and pluses in a string.

##### Examples 😃:

```
plusHashCount("###+") ➞ [1, 3]
plusHashCount("##+++#") ➞ [3, 3]
plusHashCount("#+++#+#++#") ➞ [6, 4]
plusHashCount("") ➞ [0, 0]
```

Good Luck 😀
